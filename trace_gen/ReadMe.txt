
Простой генератор алгоритма и соответствующих трасс текстовом виде
==================================================================

Главный скрипт:
	json2alg2tr.py

CMD-скрипт запуска:
	run.bat

Входные данные:
	alg_in.json	- алгоритм
	tr_in.txt	- цепочки условий для описания трасс по алгоритму (для создания разных трасс)

Выходные данные:
	algtr_out.txt	- генерируется скриптом (с перезаписью, если файл существует)
				Используйте с подсветкой синтаксиса в Notepad++
				
Настройки для подсветки синтаксиса в Notepad++ (для подсветки своих действий нужно добавить их в ключевые слова)
	Трасса_npp_v3.xml

Пример описания алгоритма (со всеми видами структур):
	alg_in-all-structs-example.json
	

Как пользоваться
============================

1. Изучить пример и вид описания алгоритма в . 
	Обратите внимание - имена полей объектов JSON важны.
	
2. Составить свой пример алгоритма в alg_in.json
3. Составить наборы условий для трасс в tr_in.txt (0 - ложь, 1 - истина)

4. Запустить run.bat

5. Открыть algtr_out.txt в Notepad++
